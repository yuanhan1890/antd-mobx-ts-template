# typescript template

使用方法:

- `git clone --depth=1 ssh://git@git.ibuscloud.com:30022/yuanpinyang/typescript-template.git <project-name>`
- `cd <project-name>`
- `rm -rf project-name`
- `git remote set-url origin <project-git-url>`
- `npm i && npm run start`
